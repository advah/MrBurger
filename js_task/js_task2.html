Задание 1:

  1. При помощи DOM API создайте DOM-элемент с тегом div
  2. Добавьте созданный элемент в DOM-дерево
  3. При помощи DOM API,
    поместите в элемент текст:
    'Этот элемент создан при помощи DOM API'

Задание 2

  1. При помощи DOM API создайте элемент с тегом div
  2. Задайте созданному элементу класс 'inner'
  3. При помощи DOM API, поместите в элемент текст:
    'Этот элемент тоже создан при помощи DOM API'
  4. Поместите созданный элемент внутрь элемента,
    который был создан в предыдущем задании

Задание 3

  1. Возьмите код из предыдущего задания
    и при помощи DOM API задайте
    созданному элементу CSS-стиль 'color: red'

Задание 4

  1. Возьмите код первого задания
  2. Добавьте обработчик события click на созданный элемент
  3. При клике на элемент, в консоль должен выводиться текст:
    'Этот текст говорит о том, что я всё сделал правильно'
  4. Откройте страничку в браузере
  5. Кликните на элемент
  6. Убедитесь, что после клика на элемент
    в консоли появилась надпись с текстом
    'Этот текст говорит о том, что я всё сделал правильно'
