Типы данных и переменные

Перечислите известные типы данных в JS:

  1. Чсловые.
  2. Строчные.
  3. Логические.
  4. Объекты.
  5. Функции.
  6. Не определенные.

  1. Создайте var-переменную с именем name и запишите в эту переменную ваше имя
  2. Выведите значение созданной переменной на экран
  3. Ниже, запишите в созданную переменную новое значение с именем вашего друга/подруги
  4. Выведите новое значение на экран
  5. Убедитесь, что в обоих случаях выводится верное значение.

Условный оператор if

  1. Напишите условный оператор if и передайте ему
    любое истинное утверждение
  2. Пусть ваш условный оператор выводит в консоль любую фразу,
    если оператору передано истинное выражение
  3. Передайте оператору любое ложное утверждение и убедитесь,
    что фраза перестала выводиться на экран
  4. При помощи else модифицируйте условные оператор так,
    чтобы в случае ложного утверждения тоже что-то выводилось на экран

Циклический оператор for:

  1. Создайте циклический оператор for.
  2. Сделайте так, чтобы его тело выполнялось 10 раз
  3. В теле цикла выведите значение счетчика

Функции:

  1. Создайте функцию sum, которая принимает три параметр: p1, p2, p3
  2. Внутри функции сложите значение трех параметров,br> и запишите результат в новую переменную
  3. Верните значение этой переменной из функции
  4. После того, как опишете функцию,
    вызовите ее и передайте ей 10, 20 и 30 в качестве аргументов
  5. Создайте переменную и результат вызовы функции запишите
    в переменную и выведите значение переменной на экран
  6. Убедитесь, что на экран выводится сумма переданных аргументов
  7. Ниже повторно проделайте вызов функции и вывод результата на экран,
    но каждый раз передавайте функции разные аргументы
  8. Убедитесь, что каждый раз выводится сумма переданных аргументов

Массивы и объекты:

Задание 1:

  1. Создайте переменную и запишите в нее массив из двух элементов ‘привет’ и ‘loftschool’
  2. Ниже, при помощи push, добавьте в этот массив еще два элемента: ‘, я изучаю’ и ‘javascript’
  3. Ниже, выедите в консоль итоговой размер массива
  4. Ниже, при помощи for выведите все элементы массива в консоль

Задание 2

  1. Создайте переменную с массивом и добавьте в этот массив 10 разных чисел.
    Пусть некоторый числа будут меньше чем 100,
    а некоторые больше чем 100. Цифры можно добавлять не по порядку
  2. При помощи цикла for переберите все числа в массиве
    и при помощи оператор if внутри тела цикла,
    выведите в консоль только те числа из массива, которые больше чем 100

Задание 3

  1. Создайте переменную и запишите в нее объект
  2. Пусть в этом объекте будут 3 свойства: name, lastName и age
  3. В свойство name запишите свое имя
  4. В свойство lastName запишите свою фамилию
  5. В свойство age запишите свой возраст
  6. Ниже обратитесь к каждому свойству и выведите значения всех свойств в консоль
  7. Ниже добавьте в объект любое новое свойство и выведите его значение на экран

Задание 4

  1. Создайте функцию с именем hello и параметром human.
  2. Напишите код вашей функции так,
    чтобы она возвращала фразу
    ‘Привет, меня зовут ИМЯ ФАМИЛИЯ и мне ВОЗРАСТ лет!’.
  3. Вместо ИМЯ нужно подставить значение свойства name из объекта,
    который вы передадите в human
  4. Вместо ФАМИЛИЯ нужно подставить значение свойства
    lastName из объекта, который вы передадите в human
  5. Вместо ВОЗРАСТ нужно подставить значение
    свойства name из объекта, который вы передадите в age
  6. Ниже вызовите созданную функцию
    и перейдите ей в качестве аргумента объект,
    который вы создали в предыдущем задании
  7. Результат вызова функции запишите
    в переменную и выведите ее значение на экран
  8. Подсказка: для того,
    чтобы объединять строки со значениями переменных,
    можно использовать оператор ‘+’,
    например ‘Привет ’ + name
